# SL Identity v1.1.0 — Dynamic Engine

## Cosa cambia
- `profile.json` diventa la fonte unica dei dati.
- `branding` separa dati aziendali e dati personali.
- `app.js` è più robusto e usa fallback se `profile.json` non viene caricato.
- `sw.js` pulisce le vecchie cache.
