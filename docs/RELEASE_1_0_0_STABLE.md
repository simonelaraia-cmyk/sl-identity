# SL Identity — Release 1.0.0 Stable

## Stato
Versione stabile del progetto dopo il completamento delle fondamenta.

## Funzioni incluse
- Card digitale responsive
- Logo e branding configurabili
- Profili multipli tramite `?id=`
- Profili in `profiles/*.json`
- Download contatto `.vcf`
- Pulsanti Chiama, Email, LinkedIn
- QR Code per il profilo corrente
- PWA / GitHub Pages
- Cache ripulita per evitare problemi su Safari
